# 4.4.2 質問システムはどう動くか

質問システムはパターンマッチングとユニフィケーションという二つの中心的演算の周囲に構築する。


## パターンマッチング

データリスト ((a b) c (a b))
パターン変数?xを(a b)に束縛
パターン(?x c ?x)とマッチ


データリスト ((a b) c (a b))
束縛
    - ?xを(a b)
    - ?yをc
    - ?zを(a b)
(?x ?y ?z)にマッチ



データリスト ((a b) c (a b))
束縛
    - ?xをa
    - ?yをb
((?x ?y) c (?x ?y))にマッチ


### 用語

データ
(job (Ben) (Computer Geek))

パターン
(job ?x (Computer Geek))

フレーム
パターン変数の束縛を規定する情報
